*Get Next Line
---

**Description
---
This activity aims to programme a function that returns a line read from a file descriptor.
Through this project, I learned an interesting new concept : static variables.

**Instructions
---
Compilation :
`gcc -Wall -Wextra -Werror`.
You can add the compiler call: `-D BUFFER_SIZE=n`.
Change n by the size of the Buffer_size wanted.
